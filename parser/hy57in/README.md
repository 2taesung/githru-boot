# parser/hy57in

git log 명령어로 콘솔에 찍은 내용을 json으로 만들기

### input

git log 명령어를 이용하여 생성된 문자열들

### output

githru에서 보여지는 필요정보들 (파일 별 수정/삭제된 라인 수 포함, json)

### challenge

파싱된 log를 commit을 node로 가지는 tree 형태의 구조체로 만들어 보세요.
